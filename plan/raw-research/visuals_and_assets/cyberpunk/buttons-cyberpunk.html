BUTTON CATALOG

SLOT.77 · cyberpunk theme · 4 types · reusable
01 · Clipped-Corner// primary CTA .btn-clip

Signature button — angled cut corners + acid glow. Reserve for the main action on a screen (START, SPIN, CONFIRM). Use 1–2 per screen max.

Default
Sizes
02 · Rect Solid// strong secondary .btn-solid

Solid-fill rectangle. Prominent but not signature — use for assertive secondary actions (APPLY, NEXT, MAX BET).

Default
03 · Rect Outlined// ghost / tertiary .btn-ghost

Transparent with thin border. For optional, supportive, or repeated actions — multiple per screen is fine (AUTO, HOW TO PLAY, CANCEL, RESUME).

Default
04 · Text// link / inline .btn-text

No border or background. For navigation, inline links, and tertiary toggles (← LOBBY, SETTINGS, VIEW MORE).

Default
HOW TO USE
· Pick 1 clipped-corner for the main CTA, then layer solid / ghost / text as hierarchy descends.
· Mix sizes freely (sm / md / lg) — the clipped-corner scales cleanly; others use padding adjustments.
· Hover to preview interactive state (glow intensifies / color shifts to acid).
· Colors come from theme variables — swap --acid to change accent without touching component.
